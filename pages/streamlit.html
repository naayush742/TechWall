Streamlit

Open-source Python library to create and share custom web apps.

History

Streamlit was founded in 2018 by Adrien Treuille, Amanda Kelly, and Thiago Teixeira, all ex-Google. The first public release was in October 2019. The concept was revolutionary: write a Python script, and Streamlit automatically creates an interactive web app — no HTML, CSS, or JavaScript required. Snowflake acquired Streamlit in March 2022 for $800 million. Streamlit Community Cloud provides free hosting for public apps.

Visual Overview

Why We Use Streamlit

Streamlit allows data scientists and ML engineers to turn Python scripts into shareable web applications in hours rather than weeks. Since the audience is Python developers, the entire framework uses Python syntax — no web development knowledge needed. Hot-reloading means apps refresh instantly as code changes.

Applications & Use Cases

  • Interactive ML model demos and prototypes
  • Data exploration dashboards
  • Internal business reporting tools
  • Research paper and experiment visualizations
  • API testing frontends
  • Computer vision demos with camera input

Key Facts & Features

  • Magic commands: st.write() handles text, DataFrames, charts, and more
  • Widgets: st.slider, st.selectbox, st.file_uploader, st.camera_input
  • Caching: @st.cache_data prevents expensive recomputation on reruns
  • Streamlit Cloud: free hosting for public GitHub repositories
  • Session state: st.session_state for multi-page apps and state management
  • st.columns and st.tabs for layout; Plotly, Altair, Matplotlib all render natively
