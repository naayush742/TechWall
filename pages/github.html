History

GitHub was founded in April 2008 by Tom Preston-Werner, Chris Wanstrath, P. J. Hyett, and Scott Chacon. It was built on Git, the version control system created by Linus Torvalds in 2005. GitHub introduced the pull request workflow that revolutionized open-source collaboration. Microsoft acquired GitHub in June 2018 for $7.5 billion. It now hosts 420+ million repositories and has over 100 million registered developers.

Visual Overview

Why We Use GitHub

GitHub is more than code storage — it is the social network for developers. It provides pull requests for code review, Issues for bug tracking, Actions for CI/CD automation, Projects for agile planning, Packages for artifact hosting, Codespaces for cloud development environments, and Copilot for AI-powered coding.

Applications & Use Cases

  • Open-source project hosting and collaboration
  • CI/CD automation with GitHub Actions
  • Code review via pull requests
  • Project and issue tracking (GitHub Projects)
  • Documentation hosting (GitHub Pages)
  • Security scanning with Dependabot and Code Scanning

Key Facts & Features

  • Git-based version control with branching and merging
  • GitHub Actions: YAML-defined automation workflows
  • GitHub Copilot: AI pair programmer trained on code
  • GitHub Pages: free static site hosting from repositories
  • Codespaces: full VS Code environment in the browser
  • GitHub Sponsors enables financial support for open-source developers
