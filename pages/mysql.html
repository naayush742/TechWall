History

MySQL was created by Michael Widenius and David Axmark and first released in May 1995. The name combines 'My' (after co-founder's daughter) and SQL. Sun Microsystems acquired MySQL AB in 2008 for $1 billion. Oracle acquired it with Sun in 2010. MySQL now powers over 50% of all websites including Wikipedia, Twitter, and YouTube. MariaDB was forked from MySQL in 2009 to remain fully open source.

Visual Overview

Why We Use MySQL

MySQL is fast, reliable, easy to install, and has decades of battle-tested stability. It uses a client-server model, supports ACID transactions via InnoDB storage engine, and integrates with virtually every programming language. It is the 'M' in the classic LAMP (Linux-Apache-MySQL-PHP) stack and remains the foundation of countless web applications.

Applications & Use Cases

  • Web application backends (WordPress, Drupal, Joomla)
  • E-commerce platforms (Magento, WooCommerce)
  • Data warehousing with MySQL partitioning
  • Embedded database in devices and applications
  • SaaS application databases
  • Logging, analytics, and reporting systems

Key Facts & Features

  • Supports all standard SQL syntax plus MySQL-specific extensions
  • InnoDB provides full ACID compliance and foreign keys
  • Replication: master-slave and group replication built-in
  • Supports JSON data type natively since MySQL 5.7
  • MySQL Workbench provides GUI for modeling and administration
  • Used by Facebook's original backend infrastructure
